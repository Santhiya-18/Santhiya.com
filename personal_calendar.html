The Problem

It has been hard to schedule meetings. Even though there are many Calendar apps used by organizations, meetings are also set outside maybe with your relative or friends. Sometimes we have "me time" or "study time". This makes it difficult to remember or coordinate meetings.

Solution

Here is the Personal Calendar app that I developed, which is very easy to use and track our plans. Even though there are many apps to track our meetings, this one directly takes the minimal required information from the user and list it in a way that can be read easily. This application is also built with a simple yet elegant design and seamless rest API which gets, reads, update and delete the data we want. It is also built with the best technologies to stay up to date. Let's see how to works!

Home Page without any data

How to add Plan?

As soon as we click on add plan button. A pop open with an efficient for where we can enter the plan name, start and end time, recurrence of the meeting such as Daily, Weekly, Monthly.

Add plan view

View

Once we have added our plan. We have two views such as All plans and Today's plan. Wherein all plan we will be able to view all the plans we have and in Today's plan we will be able to see the plans for today.

All Plan tab

How to edit plan?

Change of plan always happen. Hence, we always have an option to make a change to our existing plan. All we have to do is that click on the edit icon in the Action column against the plan we want to make a change. Then a new page with a form prepopulated with the plan data we chose appears where we can make a change of the data. Then TADA!! our list is all set.

Edit Plan view

How to delete plan?

Finally, if we don't want this particular plan we can always remove it from our list by clicking on the delete icon. Once we click on this button a popup appears with confirmation to make sure that we can proceed with the deletion.

Delete Plan view
